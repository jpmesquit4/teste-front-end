# Como rodar o projeto

## Instalação

Clone o repositório e, dentro da pasta site do projeto, instale as dependências:

```bash
npm install
```

## Rodando o projeto

Após instalar as dependências, execute o comando abaixo para iniciar o ambiente de desenvolvimento:

```bash
npm run dev
```

Após rodar o comando, aparecerá no terminal algo como:

**Local:** http://localhost:5173/

Copie esse endereço e cole no seu navegador para acessar o projeto.
